What is Haulage?

Haulage is a dependency manager for source code, written in Ruby.

It can "intelligently" retrieve dependencies defined in a JSON-based dependencies file. It supports retrieval through the APT package manager (Debian systems) as well as standard HTTP and naturally Ruby Gems.

Haulage is powered by the haul Ruby module, which you can use in your own scripts or programs. API modules exist for Python and Perl [WIP] as well as Ruby.

You can clone Haulage from its Git repo, using: 
git clone git://github.com/stpettersens/Haulage
